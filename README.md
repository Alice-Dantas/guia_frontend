
# Guia Frontend

Este repositório reúne meus estudos em **desenvolvimento frontend**, organizados de forma progressiva e modular, abrangendo desde os fundamentos da web até animações e gráficos 3D no navegador.

## Material disponível

Apostilas e exemplos organizados por tecnologia, com foco nos fundamentos do desenvolvimento frontend, abordando:

### HTML
- Documento HTML (anatomia).
- Elementos HTML (tudo sobre as principais tags).

### CSS
- Fundamentos da estilização de páginas web (seletores, propriedades básicas, unidades de medida).
- Modelo de Caixa (Box Model).
- Layout e Posicionamento.
- Tipografia e Texto.
- Cores e Gradientes.
- Imagens e Fundos.
- Animações e Transições.
- Responsividade.

### JavaScript
- Variáveis e Tipos; Tratamento de Dados; Operadores.
- DOM (Document Object Model).
- Eventos.
- Estruturas Condicionais e de Repetição.
- Tipos de Escopo.
- Funções.
- Promisse.
- Spead Operator.

### GSAP
- Conceitos básicos da biblioteca GSAP.
- Criação de animações controladas e performáticas.
- Uso de timelines para sequenciamento de animações.
- Exemplos práticos de animações aplicadas a interfaces web.

### Three.js
- Introdução à biblioteca Three.js.
- Conceitos fundamentais para criação de cenas 3D no navegador.
- Estrutura básica de uma cena (câmera, luz, objetos).
- Exemplos práticos de aplicações e experimentos em 3D.

### Exemplos de código
- Implementações práticas dos conceitos abordados em cada tecnologia.
